🔐 Secure Password Generator

A modern and responsive Secure Password Generator built using HTML, CSS, and JavaScript. This web application allows users to generate strong, random, and customizable passwords to improve online security. Users can choose the password length and include uppercase letters, lowercase letters, numbers, and special characters. The application also provides a password strength indicator and a one-click copy-to-clipboard feature.

✨ Features
🔑 Generate secure random passwords
📏 Choose custom password length (4–50 characters)
🔠 Include uppercase letters
🔡 Include lowercase letters
🔢 Include numbers
🔣 Include special characters
📋 Copy password to clipboard with one click
💪 Password strength indicator (Weak, Medium, Strong)
🎨 Modern Glassmorphism user interface
📱 Responsive design for desktop and mobile devices
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and Glassmorphism design
JavaScript (ES6) – Password generation logic and interactivity

🚀 How to Run
Download or clone this repository.
Open the project folder in Visual Studio Code.
Open index.html in your browser or use the Live Server extension.
Choose your password preferences.
Click Generate Password to create a secure password.
Click Copy to copy the generated password to your clipboard.
